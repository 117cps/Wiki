# Beacon of the Void

![etherte](https://vwiki.valorserver.com/api/item/picture/beacon%20of%20the%20void)

Effect on Self: Bravery for 3 seconds

Banner: Within 7 sqrs Empower allies for 7 seconds | Stays active for 6 seconds

MP Cost: 95

Cooldown: 7 seconds

    +8 defense

Fame Bonus: 8%
