# Shine

![shine](https://vwiki.valorserver.com/api/item/picture/shine) 

Shine cost without shine booster: 75,000

Shine cost with shine booster: 70,000

*Rerolls any LG item with the text "Can be used with shine" on it into a random shine item. (shine has the exact same legendary pool as the shard of sidon's fall click on the shard Logo to view in shard showcase)*
