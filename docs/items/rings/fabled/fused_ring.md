# Fused Ring

![Fusedr](https://vwiki.valorserver.com/api/item/picture/fused%20ring)

    +12 Vitality
    
    +12 Wisdom
    
    +2 Mana Leech
    
Fame Bonus: 6%

**Drops from the Twilight Archmage of the Shatters**
